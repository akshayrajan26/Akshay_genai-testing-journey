# LLM Test Utils

This is the Week 4 mini-project for the GenAI Testing Journey. 

## Overview
This package bundles various utilities developed in Weeks 1-3 into a reusable Python package.

## Features
- **Formatters**: Dynamic prompt template handling.
- **Validators**: Logic to verify LLM responses.
- **Test Case Manager**: Organize and iterate through test sets.

## Installation
You can install this package locally using:
```bash
pip install -e .
```

## Usage
Run the `main.py` script to see a demonstration of the package in action:
```bash
python main.py
```
